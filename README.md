# AI Interview Platform

An AI-powered interview preparation platform that uses Google Gemini to generate interview questions, simulate real interview sessions, and provide feedback to help users improve their skills.

---

## 🚀 Features
- AI-generated interview questions (Gemini powered)
- Role-based interview simulation
- Real-time answer evaluation
- User authentication system
- Session tracking and history
- Full-stack MERN architecture

---

## 🧠 Tech Stack
- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MongoDB
- AI Model: Google Gemini API

---

## 📁 Project Structure
- backend/ → API, controllers, models, routes
- frontend/ → React UI
- database-config.js → DB connection setup

## ⚙️ Setup Instructions

### 1. Clone the repo
git clone https://github.com/Hemant-Gith/Ai-Interview-Platform-using-Gemini


### 2. Install backend dependencies

cd backend
npm install


### 3. Install frontend dependencies

cd frontend
npm install


### 4. Setup environment variables
Create `.env` file in backend:


MONGO_URI=your_mongodb_connection
GEMINI_API_KEY=your_gemini_api_key


### 5. Run the project
Backend:

npm run dev


Frontend:

npm run dev
